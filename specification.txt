Bootstrap:
Bootstrap was used to create a responsive navigation bar, styled buttons, tables, containers, and form controls. These components help ensure the website is visually appealing and responsive across desktops, tablets, and mobile devices.

JavaScript:
JavaScript was used to add interactivity to the website in two ways:

1. A Fun Facts Generator displays a random technology-related fact each time the user clicks a button on the homepage.
2. A Contact Form validates user input by ensuring that the Name, Email, and Message fields are completed and that the email address follows a valid format. If the form is completed correctly, a personalized confirmation message is displayed; otherwise, an error message is shown.
